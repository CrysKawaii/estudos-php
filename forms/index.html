Formulários

Os formulários são uma das coisas mais comunus da internet hoje. Quase qualquer site que você entra, tem que preencher um formulário, tanto para se cadastrar ou apenas colocar seu e-mail para receber avisos de determinado assunto que te interessa.

Um formulário não pode ser feito apenas com HTML, ele precisa de uma linguagem de programação para ter a interação com o usuário, não adianta preencher com os dados se esses dados não serão utilizados para nada. Eh aí que entra a parte do PHP em formulários da internet.

Com o PHP os dados podem ser enviados e recebidos para fazer qualquer tarefa necessária com os dados, como por exemplo receber o nome e sobrenome de alguém e exibir uma mensagem de boas vindas para essa pessoa. Eh isso que será feito nos exemplos abaixo.

Veja este simples formulário, ele apenas pede o nome e sobrenome de alguém, esse dados quando enviados irão para um arquivo em PHP que receberá do HTML os valores (value) enviados pelo usuário e processados para retornarem alguma resposta, qualquer que seja.

Este segundo formulário faz a mesma coisa, porém com um pequeno detalhe, ambos os formulários tem métodos de envio diferentes. O primeiro eh com o método 'get', já esse segundo eh com o método 'post'. Para pegar os dados enviados é necessário colocar em uma variável através do métdodo. Por exemplo o formulário de cima usa o método get, então eu faço: $nome = $_GET['nome'], o que eu acabei de fazer foi basicamente pegar o elemento com o name = nome e colocar ele dentro da variável através do método get. O segundo formulário é a mesma coisa, mas com o método post.

Qual a diferença entre os dois? Bom, o método 'get' é geralmente usado para dados que não são sensiveis, isso se deve ao fato de que com esse método os valores enviados ficam a mostra para qualquer um que olhe na URL do site, por isso não é tão recomendado se usar para receber senhas por exemplo. Já o método 'post' não deixa tão a mostra esse dados, porém não é como se eles estivessem 100% seguros de qualquer coisa, se o usuário procurar um pouco mais, ele facilmente encontra. Essa questão de segurança é mais para frente com MySQL.

E ainda tem um terceiro método, o $_REQUEST que recebe qualquer valor(value) de qualquer métedo, sendo ele do $_GET ou $_POST, vai funcionar. Essa variável é uma junção da $_GET, $_POST e $_COOKIES, por isso funciona em ambos os casos.

São váriaveis superglobais as $_GET, $_POST e $_REQUEST, começou com $(cifrão) é variável em PHP. São chamadas de superglobais porque já são variávei declaradas no PHP, então independente do programa, ela vai funcionar.
